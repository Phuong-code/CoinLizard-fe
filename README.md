# React JS + Vite


### Clone to local

```bash
git clone https://github.com/Phuong-code/CoinLizard-fe.git
pnpm install # If you don't have pnpm installed, run: npm install -g pnpm
```

## Usage

### Development

```bash
pnpm dev
```

### Build

```bash
pnpm build
```

And you will see the generated file in `dist` that ready to be served.

### Serve dist

```bash
pnpm serve
```
